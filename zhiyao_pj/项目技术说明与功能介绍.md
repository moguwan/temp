# MediSyn 制药工程 项目技术说明与功能介绍

## 一、项目概述

本项目为 **MediSyn 制药工程** 展示网站，围绕**口-脑轴**与阿尔茨海默病（AD）预防，展示工程化益生菌、口腔健康与 AD 关系的研究成果。项目采用纯静态站点架构，无后端服务，适合静态托管部署。

---

## 二、技术栈

### 2.1 前端技术

| 类别 | 技术 | 说明 |
|------|------|------|
| **核心** | 原生 HTML5 + CSS3 + JavaScript | 无前端框架，轻量级实现 |
| **图标库** | Font Awesome 6.0.0 | 通过 CDN 引入，用于导航、按钮等图标 |
| **布局** | Flexbox + Grid | 响应式布局，适配不同屏幕 |
| **动画** | CSS Transition + cubic-bezier | 页面切换、悬停等过渡效果 |

### 2.2 架构模式

| 特性 | 实现方式 |
|------|----------|
| **SPA 形态** | 主壳 `app.html` + iframe 子页，横向滑动切换 |
| **导航** | 固定顶部导航栏 + 点击切换页面 |
| **跨 iframe 通信** | `postMessage` 传递 `{ type: 'app-navigate', page: 'xxx' }` |

### 2.3 资源与托管

| 资源类型 | 方案 |
|----------|------|
| **图片** | 图床 pic1.imgdb.cn 托管 |
| **构建工具** | 无（无 webpack、vite、npm 等） |
| **部署** | 静态文件直接部署，适合 GitHub Pages、Nginx 等 |

### 2.4 辅助工具（Python）

| 脚本 | 功能 | 依赖 |
|------|------|------|
| `compress_images.py` | 图片压缩：>10MB 的 PNG 转 JPG，SVG 转 PNG | Pillow (PIL) |
| `build_tuchuang.py` | 收集网页引用图片，去重排序后复制到图床素材目录，生成 `图床素材清单.md` | 标准库（hashlib、shutil） |

## 三、主要功能

### 3.1 横向滑动 SPA

- 9 个页面横向排列，总宽度 900vw
- 使用 `transform: translateX()` 切换页面
- 过渡动画：`cubic-bezier(0.25, 0.46, 0.45, 0.94)`，约 0.7s
- 顶部固定导航栏，点击按钮切换对应页面

### 3.2 通用交互功能

| 功能 | 说明 |
|------|------|
| **嵌入模式检测** | 子页在 iframe 中时隐藏导航，点击转发给父窗口 |
| **折叠菜单** | `.toggle-btn` 控制 `.collapse-content` 展开/收起 |
| **平滑滚动** | 锚点链接平滑滚动到对应章节 |
| **Scroll Spy** | 侧边导航随滚动高亮当前章节 |
| **返回顶部** | 滚动到一定位置后显示返回顶部按钮 |
| **实验折叠** | `.experiment-box` 点击展开/收起 |
| **手风琴** | `.hp-accordion-item`、`.docx-accordion` 点击展开/收起 |
| **弹窗** | 成员介绍页 `awardModal` 的打开/关闭 |
| **首页圆形导航** | 左侧 5 个圆形按钮对应 5 个 section，支持点击和滚动高亮 |

### 3.3 文档嵌入

- 部分页面支持 iframe 嵌入 PDF/Word 文档（如实验设计文档、调查问卷报告、质粒设计文档等）
- 通过折叠面板（accordion）形式展示，点击标题展开查看

## 四、页面结构与内容

### 4.1 入口与主壳

| 文件 | 说明 |
|------|------|
| `index.html`（根目录） | 根入口，自动跳转到 `01-code/app.html` |
| `01-code/app.html` | 主应用壳，包含固定导航栏和 9 个 iframe 子页的横向滑动容器 |

### 4.2 页面列表与内容

#### 1. 首页 (`01-code/首页/index.html`)

**主题**：口-脑轴侦探调查，拦截阿尔茨海默病

**内容板块**：
- **开篇引入**：记忆消逝的开端与口腔健康的关系
- **科学背景**：口-脑轴理论、牙周炎与 AD 的关联
- **传统方案痛点**：现有治疗手段的局限性
- **产品方案**：MediSyn 工程化益生菌 Agent EcN 的解决方案
- **愿景升华**：项目愿景与展望

**特色**：左侧圆形导航对应 5 个大卡片，支持点击跳转和滚动高亮；封面图采用「案件档案」风格设计。

---

#### 2. 项目介绍 (`01-code/项目介绍/description.html`)

**主题**：MediSyn 品牌与项目全貌

**内容板块**：
- MediSyn 品牌介绍
- **项目背景**：口腔健康现状、危害因素评估
- **病理机制**：抗生素的局限性、决定制作工程化益生菌
- **机制探究**：融化/剪切/杀灭、中和毒素
- **影响与风险**：构建质粒、瓦解效率评估
- **预防与治疗**：公众认知→预防、日常护理→治疗
- 项目摘要与总结

**特色**：侧边导航含多级子菜单，支持 Scroll Spy 高亮。

---

#### 3. 项目设计 (`01-code/项目设计/engineering.html`)

**主题**：工程化 Agent EcN 的设计思路

**内容板块**：
- **概述**：工程化思维、Research-Design-Build-Test-Learn 循环
- **循环一：核心武备**：DspB、DNase I、Cas9、Cystatin C 等效应蛋白设计
- **循环二：底盘筛选**：大肠杆菌底盘选择
- **循环三：破防协同**：生物膜瓦解与杀灭协同
- **循环四：生物安全与靶向**：安全性与靶向性设计
- 参考文献

**特色**：图文并茂展示战术概念图、基因线路图等。

---

#### 4. 线路与元件 (`01-code/线路与元件/result.html`)

**主题**：质粒与基因线路设计

**内容板块**：
- **质粒设计＆模拟克隆过程**：可折叠文档查看器
- **质粒设计**：质粒结构示意图
- **线路设计**：基因线路图
- **产品设计**：产品形态与设计图

**特色**：支持 iframe 嵌入质粒设计文档，折叠面板展示。

---

#### 5. 实验设计 (`01-code/实验设计/experiment.html`)

**主题**：实验方案与干实验模拟

**内容板块**：

**特色**：突出「干实验模拟与评估」板块，采用醒目样式；图片网格展示实验图。

---

#### 6. 人类实践 (`01-code/人类实践/hp.html`)

**主题**：公众认知调研与人类实践

**内容板块**：
- **人类实践概况**：实践活动图片展示
- **调查问卷报告**：关于口腔健康与 AD 预防的公众认知调研报告（可折叠文档查看器）

**特色**：结合 hp 文档中的调研报告，展示公众对口腔健康与 AD 预防的认知情况。

---

#### 7. 建模 (`01-code/建模/model.html`)

**主题**：DBTL 循环中的建模工作

**内容板块**：

- **生物膜瓦解动力学**：动力学模型
- **剂量-效应优化**：剂量与效应关系
- **Conclusion**：建模结论

**特色**：采用图文左右/上下布局的模块化展示，每个模块含标题、简介、关键元素、关键洞察等。

---

#### 8. 成员介绍 (`01-code/成员介绍/personal.html`)

**主题**：团队与品牌形象

**内容板块**：
- **菌菌**：MediSyn 品牌形象大使

**特色**：卡片式布局，悬停动效；无侧边导航，无滚动提示。
